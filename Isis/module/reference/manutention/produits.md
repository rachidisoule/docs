# Produits

Cette option permet de gérer les produits.

![](../../../.gitbook/assets/Produit-home.PNG)

**Edition de la fiche : Produits**

**NB :** Seule les zones en astérisque (\*) de cet écran sont obligatoire.

* **Code:** Indiquez le code
* **Libellé :** Indiquez le libellé
* **TypeProduit :** Indiquez le type de produit
* **GroupeProduit :** Indiquez le groupe de produit
* **Vrac:** Indiquez si le produit est de type Vrac
* **Suivi:** Indiquez si le produit est de Suivi
* **Commentaire :** Indiquez le commentaire
